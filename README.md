## Sobre mim

Sou desenvolvedor fullstack e fundador da **XzinTech** — construo sistemas reais para resolver problemas reais de negócios.

Nos últimos meses coloquei em produção um ERP para concessionárias, um SaaS multi-tenant para gráficas e uma plataforma de gestão de apostas. Não faço CRUD genérico: cada sistema tem lógica de negócio complexa, está em produção e é usado por clientes reais.

Stack principal: **Node.js · Express · React · PostgreSQL · Supabase · Railway**

---

## 🚀 Projetos em Produção

| Projeto | Descrição | Stack |
|---|---|---|
| 🏍️ **MotoNow Gestão** | ERP completo para concessionárias Shineray — estoque, vendas, comissões, multi-filial, oficina | React · Node.js · PostgreSQL · Railway |
| 🖨️ **XzinTech Gráficas** | SaaS multi-tenant para gráficas — pedidos, clientes, financeiro, planos de assinatura | Node.js · Supabase · JWT · Railway |
| 📊 **XzinBets** | Plataforma de gestão de apostas — value bets, bankroll, tipsters, streak alerts | React · Supabase · Express · Recharts |
| 🤖 **Chatbot Shineray** | Automação WhatsApp para atendimento, captação de leads e simulações de financiamento | WhatsApp API · Node.js |
